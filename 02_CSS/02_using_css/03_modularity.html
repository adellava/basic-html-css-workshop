Modularity

Il CSS contiene regole di stile che sono globali by design.

E' tuttavia possibile scomporre regole di stile in file separati in modo da organizzarle meglio e rendere il codice più leggibile.

            
@import "./link-element-example.css";
@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

Nel capitolo 2.1.2 sui colori e font abbiamo già usato questa scrittura che possiamo usare anche per i nostri file.

Oppure importando più file attraverso

            
<link href="./link-element-example.css" rel="stylesheet" >

E' importante sapere che...

A. usando più link i file css vengono scaricati in parallelo

B. usando un unico link dove importo più file questa cosa non avviene
