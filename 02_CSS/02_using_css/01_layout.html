Il layout è l'architettura dello spazio. Esso si occupa di organizzare lo spazio per i componenti e di stabilire i confini dei loro comportamenti come lo scrolling o l'utilizzo dell'asse z nella pagina.
Spesso in un design system esistono blocchi che si occupano unicamente di organizzare il layout.

Si può parlare anche, metaforicamente, di "layout interno" o di "layout del componente", quando si organizza lo spazio specifico di un componente.

Le due proprietà di stile che maggiormente determinano il layout sono:

  • position: che di default è static, specifica se un elemento deve essere posizionato o meno all'interno del page flow
  • display: che di default è static,

1. layout grid

Grid è pensato per creare layout in 2 dimensioni.

                
/* Grid Layout */

.grid {
    display: grid;
    grid-template-columns: 200px auto;
    grid-template-rows: minmax(30px, 50px) 1fr minmax(30px, 50px);
    grid-column-gap: 10px;
    grid-row-gap: 20px;
    grid-template-areas: 
    "header header"
    "aside  maincontent"
    "footer footer";
    width: 100%;
    height: 100%;
}

.grid__header {
    grid-area: header;
    background-color: red;
}

.grid__footer {
    grid-area: footer;
    background-color: blue;
}

.grid__aside {
    grid-area: aside;
    background-color: green;
    overflow: auto;
}

.grid__maincontent {
    grid-area: maincontent;
    background-color: gray;
    overflow: auto;
    -webkit-overflow-scrolling: touch;
}

2. layout flex

Flex è pensato per creare layout in 1 dimensione, per questa ragione è più appropriato utilizzarlo per i layout interni dei component oppure per disporre elementi dello stesso tipo (es. card).

                
/* Flex Layout */

.flex-horizontal {
    display: flex;
    width: 100%;
    height: 100%;
    flex-direction: row;
    position: absolute;
}

.flex-horizontal__aside {
    flex: 0 0 200px;
}

.flex-horizontal__body {
    flex: 1 1 auto;
    position: relative;
}

.flex-vertical {
    position: absolute;
    display: flex;
    width: 100%;
    height: 100%;
    flex-direction: column;
}

.flex-vertical__bar {
    flex: 0 0 60px;
    overflow-x: hidden;
    overflow-y: auto;
}

.flex-vertical__body {
    flex: 1 1 auto;
    position: relative;
}

.scrollable {
    overflow: auto;
    -webkit-overflow-scrolling: touch;
}

/* Flex Elements */

.flex-container {
    display: flex;
    width: 100%;
    justify-content: space-around;
    flex-flow: row wrap;
}

.flex-container__child {
    width: 150px;
    height: 200px;
    margin: 0 auto 20px auto;
    background-color: yellow;
}
