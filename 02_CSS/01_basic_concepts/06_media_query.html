Media Query

Le media query possono porre regole di stile condizionate, ecco le principali categorie:

  • tipo di supporto media: screen, print, speech
  • grandezza del viewport: max-width, min-width, max-height, min-height
  • ma anche densità di pixel: resolution
  • orientamento e aspect ratio del viewport: aspect-ratio, orientation
  • capacità del device di supportare determinati comportamenti: hover, any-hover, pointer, any-pointer e come sperimentali scripting, update ancora poco supportate
  • le regole possono essere combinate con operatori logici come: and ',' NB ',' funziona come un or

NB per una spiegazione esaustiva leggere la documentazione

            
.rect {
    background-color: darkslategrey;
    height: auto;
    min-height: 40px;
    color: white;
    margin: 0 auto;
    width: 800px;
    text-align: left;
}
.wrapper {
    margin: 10px;
    width: calc(100% - 20px);
    background-color: #eee;
    text-align: center;
    padding: 20px 0;
}
@media screen and (max-width: 1024px) and (orientation: landscape) { 
    .rect {
        width: 100%;
    }
    .wrapper {
        background-color: #ccc;
    }
}

solitamente sono margin: 0 auto; width: 800px; text-align: left;

se @media screen and (max-width: 1024px) and (orientation: landscape) allora sono width: 100%;
