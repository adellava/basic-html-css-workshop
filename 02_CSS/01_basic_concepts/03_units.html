Units

1. Principali unità Assolute

                
.px {
    width: 300px;
}
.mm {
    width: 300mm;  /* usato principalmente per i css di stampa */
}
sono largo 300px
sono largo 300mm

2. Principali unità Relative

                
.wrapper {
    font-size: 28px;
}
.perc {
    width: 20%; /* 20% della stessa proprietà nel padre */
}
.em {
    width: 5em; /* 5 volte il valore di larghezza della lettare M col font size del padre */
}
.rem {
    width: 5rem; /* 5 volte il valore di larghezza della lettare M col font size del root element */
}
.vw {
    width: 20vw; /* 20/100 del viewport width */
}
.vh {
    width: 20vh; /* 20/100 del viewport height */
}
.vmin {
    width: 20vmin; /* 20/100 del valore più piccolo tra viewport width e viewport height */
}
.vmax {
    width: 20vmax; /* 20/100 del valore più grande tra viewport width e viewport height */
}
.ch {
    width: 5ch; /* 5 volte la larghezza del carattere "0" */
}

NB l'uso di calc in .wrapper => width: calc(100% - 20px);
sono largo 20%
sono largo 5em
sono largo 5rem
sono largo 20vw
sono largo 20vh
sono largo 20vmin
sono largo 20vmax
00000 sono largo 5ch
