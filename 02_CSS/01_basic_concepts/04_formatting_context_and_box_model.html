Formatting Context & Box Model

1. Formatting Context

Il formatting context è un area di spazio dove, all'interno di un layout di un elmento block, interagiscono anche elementi di tipo float.
questo può portare a delle problematiche di cattiva gestione dell'overflow nel suo contenimento, come si gestisce? 

                
.blockwrapper {
    background-color: #eee;
    border: 1px solid #00f;
}
.floatrect {
    float: left;
    width: 100px;
    height: 100px;
    background-color: #f00;
}
.overflowauto {
    overflow: auto;
}
.displayflowroot {
    display: flow-root;
}
.clear {
    content: "."; 
    visibility: hidden; 
    display: block; 
    height: 0; 
    clear: both;
}

2. Box Model

    background-color: #1ff;
    width: 200px;
    height: 200px;
    padding: 20px;
    margin: 20px;
    box-sizing: border-box;
}
.box2 {
    background-color: #f1f;
    width: 200px;
    height: 200px;
    padding: 20px;
    margin: 20px;
    box-sizing: content-box;
}

    background-color: #ff1;
