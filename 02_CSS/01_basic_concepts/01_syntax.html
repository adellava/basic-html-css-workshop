1. Sintassi Base

Tutto il il codice CSS è formato da regole di stile descrittive. Ogni regola è composta da uno o più selettori e di proprietà.

                
selettore1, selettore2 {
    // lista di proprietà che il codice attribuisce ai selettori
    // ogni proprietà segue la seguente sintassi
    // "nome-proprieta: valore;"
    // ci sono un numero sterminato di proprietà che è impossibile coprire in un corso così breve
    // meta esempi
    colore: rosso;
    bordo: 1px tratteggiato nero;
    sfondo: grigio;
}

esempio

                
header {
    color: blue;
}

2. Selettori

Ogni selettore individua una parte di DOM da regolare. Si può fare un selettore in diversi modi:

                
/* selettori di tipo elemento */
article {
    color: purple;
}

/* selettori di attributo */
[data-mio-attributo] {
    color: darkcyan;
}

/* selettore di classe */
.miaclasse {
    color: blue;
}

/* selettore id */
#mioid {
    color: coral;
}
<article />
<div data-mio-attributo >attributo</div>
<div class="miaclasse" >classe</div>
<div id="mioid" >id</div>
  1. selettore di tipo <elemento>
  2. selettore di attributo <li >
  3. selettore di classe <li class="miaclasse" >
  4. selettore di id <li id="mioid" > (NB ogni id è unico all'interno della pagina)

NB => 0. selettore universale * da usare con estrema parsimonia

2.1 Selettori combinati

si possono combinari due selettori o più selettori con:

                
/* elemento figlio */
.mialista .classefiglia {
    color: red;
}

/* elemento figlio diretto */
.mialista > .classefiglia {
    color: teal;
}


/* due classi "insieme" */
.classefiglia.classefiglia-modificata {
    color: yellowgreen;
}

/* elemento adiacente */
.mialista ~ .mianota {
    color: brown;
}

/* elemento subito adiacente */
.mialista + .mianota {
    color: purple;
}

<p class="mianota" >elemento precedente</p>

  1. <li > elmento di lista con .classefiglia in un <span class="classefiglia" >sotto elemento</span> </li>
  2. <li class="classefiglia" > elmento di lista con la classe </li>
  3. <li > elmento di lista con .classefiglia e .classefiglia-modificata <span class="classefiglia classefiglia-modificata" >insieme</span> </li>
  4. <li > elmento di lista senza la classe </li>

<p class="mianota" >primo elemento successivo, subito adiacente</p>

<p class="mianota" >secondo elemento successivo</p>

  1. elementi figli' ', esempio .mialista .classefiglia
  2. elementi figli diretti' > ', esempio .mialista > .classefiglia
  3. elementi adiacenti' ~ ', esempio .mialista ~ .mianota
  4. elementi subito adiacenti' + ', esempio .mialista + .mianota

NB Coi selettori combinati è facile scrivere codice poco leggibile e mantenibile in quanto non è sempre evidente il grado di specificità.

E' bene usarli con attenzione.

2.2 Specificità dei Selettori

In caso di conflitto fra regole di stile vince quella con specificità più alta

la specificità è un numero calcolato in base al tipo di selettori usati. Vince il selettore col numero più alto

come si calcola?

  • +1 per ogni selettore di elemento
  • +10 per ogni selettore di classe
  • +100 per ogni selettore id
  • 1000 per tutte le proprietà espresse in line
  • a parità di specificità vince quello scritto per ultimo

esempi:
.miaclasse = 10
.miaclasse elemento = 11
.contenitore .miaclasse = 20
#contenitore .miaclasse = 110

è un qualcosa che può generare problemi se mal gestito? spoiler: sì
esempio di bad smell: 

                    
.testo {
    font-size: 13px;
    font-style: italic;
    text-decoration: underline;
}

.contenitore {
    border: 1px solid red;
    padding: 10px;
    background-color: #ddd;
}

.contenitore .testo {
    font-size: 9px;
    text-decoration: none;
}

#testo-speciale {
    font-size: 24px;
    text-decoration: overline;
}

3. Ereditarietà

Le proprietà vengono ereditate dagli elementi figli fino a che non vengono sovrascritti

                
.root {
    color: blue;
}
#idesempio {
    color: red;
}

4. Variabili

Le variabili in CSS esistono nativamente e possono essere globali o legate a un selettore

                
                    
:root { --color: blue; }
#idesempio { --color: red; }

.root {
    color: var(--color);
}
#idesempio {
    color: var(--color);
}
