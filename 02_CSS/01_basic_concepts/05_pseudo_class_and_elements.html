Pseudo class & pseudo elements

1. Pseudoclass

Le psudo class permettono di scrivere regole di stile per stati di un determinato elemento. Sono validi esempi:

  • la codizione di :hover di un link o un bottone
  • la codizione di :invalid di un campo di input
  • la codizione di :only-child rispetto a un elemento padre
  • [...] per leggerli tutti vai qui
                
.miobottone {
    border: none;
    margin: 0;
    padding: 10px 20px;
    min-width: 5ch;
    background: #eee;
    color: #000;
    line-height: 1;
    cursor: pointer;
}

.miobottone:hover {
    background-color: #bae8ff;
}

1. Pseudoelements

Gli psudo elments permettono di scrivere regole di stile per "sottoparti" di un elemento, anche non precedentemente visibili. Sono validi esempi:

  • la codizione di ::first-line la prima linea di un elemento
  • le condizioni ::after, ::before sono un po' diversi, creano elementi che si affiancano a quello selezionato
  • [...] per leggerli tutti vai qui
                
.mioparagrafo {
    padding: 10px 20px;
    background: #eee;
    color: #555;
    text-indent: 20px;
}

.mioparagrafo::first-line {
    color: #000;
    text-transform: uppercase;
}

.mioparagrafo::first-letter {
    font-size: 40px;
    font-family: Georgia, 'Times New Roman', Times, serif;
}

.mioparagrafo::after {
    content: "Alessandro Barbero";
    display: block;
    margin-top: 20px;
}
